Weekly Timesheet

Submit every Friday for the full work week (Monday to Friday). Smallest unit: 0.0625 days = 30 min (0.125 = 1h, 0.25 = 2h, 0.5 = half day, 1 = full day). Standard week: 5 days. If less than 5, the gap auto-fills Autre (uncategorized). If more than 5, you'll see a warning but can still submit (overtime is recorded). Resubmit anytime, the latest submission overrides the previous one for the same week.
